控件的使用

控件库依赖jquery@1.9+库和bootstrap@3+ 库，请务必在使用前确保这两个库安装完成

第一步:

下载安装nxdc（需要预先安装bower）

		
			bower install http://git.dy/vigo.feng/nxdc-component-library.git
第二步:

页面嵌入nxdc.min.js 和 nxdc.min.css (假设上一步把nxdc库安装到了lib目录下)

		
			<script type="text/javascript" src="lib/NDPUI/com/nxdc.min.js"></script>
		
	
		
			<link  rel="stylesheet" src="lib/NDPUI/com/css/nxdc.min.css" type="text/css">
确保安装完成后的页面结构如下
		
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />
    <meta content="width=device-width,initial-scale=1.0,maximum-scale=1.0,user-scalable=1;" name="viewport" />
    <meta content="yes" name="apple-mobile-web-app-capable" />
    <meta content="black" name="apple-mobile-web-app-status-bar-style" />
    <meta name="format-detection" content="telephone=yes" />
    <meta content="false" name="twcClient" id="twcClient" />
    <link href="lib/bootstrap/dist/css/bootstrap.min.css" type="text/css" rel="stylesheet">
    <link href="lib/NDPUI/com/css/nxdc.css" rel="stylesheet" type="text/css">
    <script type="text/javascript" src="lib/NDPUI/com/nxdc.min.js"></script>
    <!--[if lt IE 9]>
		<script src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script>
		<script src="//cdn.bootcss.com/respond.js/1.4.2/respond.min.js"></script>
	<![endif]-->       
    <title>NDP media web ui lib</title>
</head>
<body>
	   	 	   	  	   	 	   	  
</body>
</html>
第三步：

大部分组件的使用都需要预先在DOM 中插入一个wrapper节点。例如：需要在页面中生成一个列表组件

首先在dom中插入一个节点
		
<div class="ndp-table-wrapper" name="plain"></div>
然后调用相关插件，实例化组件
		
$(".ndp-table-wrapper[name='plain']").table({
	head:{col1:"第一列",col2:"第二列",col3:"第三列",col4:"第四列"},
	data:[
		{id:1101,name:"hello123",info:"hello -china 123",gender:"man"},
		{id:1102,name:"hello_23",info:"hello - japan 34",gender:"woman"},
		{id:1103,name:"hello_34",info:"hello - euro 23",gender:"no sign"}
	]	
});

特别注意：以下几个组件不需要预先在dom上预设wrapper节点，直接使用即可

tip, alert, confirm, prompt
